# 画像の配置

以下の画像をこのフォルダに配置すると、サイトに表示されます。

| ファイル名 | 表示場所 |
|-----------|----------|
| `web-fanclub-pc-1@2x.png` | 会員プランセクション「Webファンクラブ」のカード画像 |
| `web-fanclub-pc@2x.png` | 会員特典セクション「アプリファンクラブ」のカード画像 |
| `Frame4.svg` | 会員プランセクション「入会する」見出しの左側装飾 |
| `Frame5.svg` | 会員プランセクション「入会する」見出しの右側装飾 |

※ Anima のエクスポートで元の画像ファイル名が失われていたため、ローカル参照に変更しています。デザインの元画像があれば、上記のファイル名で保存してこのフォルダに置いてください。

※ **Frame4.svg / Frame5.svg** は、デスクトップの「ファンクラブ Img PC」フォルダにある `Frame 4.svg` と `Frame 5.svg` を、それぞれ `Frame4.svg`・`Frame5.svg` の名前でこのフォルダにコピーしてください。
